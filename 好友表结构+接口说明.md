# 好友表结构 + 接口说明

## 一、friends 表结构（SQLite）

```sql
CREATE TABLE IF NOT EXISTS friends (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  user_id    TEXT NOT NULL,       -- 用户 open_id
  friend_id  TEXT NOT NULL,       -- 好友 open_id
  group_id   TEXT,                -- 来源小组ID（可选）
  group_name TEXT NOT NULL DEFAULT '', -- 来源小组名称
  created_at TEXT NOT NULL DEFAULT (datetime('now')),
  UNIQUE(user_id, friend_id),    -- 不可重复添加同一好友
  FOREIGN KEY (user_id) REFERENCES users(open_id)
);
```

**索引**：
```sql
CREATE INDEX IF NOT EXISTS idx_friends_user  ON friends(user_id);
CREATE INDEX IF NOT EXISTS idx_friends_friend ON friends(friend_id);
```

**自动建立好友逻辑**：
- 创建固定小组时 → 创建者与所有现有成员互为好友
- 加入有 `group_id` 的房间时 → 新加入者与房间内所有成员互为好友

## 二、好友/麻神相关接口列表

### 1. GET /api/v1/ranking?type=today|total
**需要鉴权**：是（JWT）
**功能**：获取当前用户的好友排行
**参数**：
| 参数 | 类型 | 说明 |
|------|------|------|
| type | string | `today`=今日排行，`total`=总计排行 |

**响应示例**：
```json
{
  "items": [
    {
      "userId": "oXXX...XXX",
      "nickname": "麻将达人",
      "avatarUrl": "https://...",
      "score": 45.5,
      "groupName": "周五固定局",
      "isMe": true
    }
  ],
  "type": "today"
}
```

---

### 2. POST /api/v1/ranking/friend/delete
**需要鉴权**：是（JWT）
**功能**：删除好友（双向删除）
**请求体**：
```json
{ "friendUserId": "oXXX...XXX" }
```
**响应示例**：
```json
{ "code": 0, "message": "删除成功", "changes": 2 }
```

## 三、前端调用封装（services/ranking.js）

```js
const { request } = require('../utils/request');

// 获取麻神排行（type: 'today' | 'total'）
function getRanking(type) {
  return request({ url: '/ranking', data: { type } });
}

// 删除好友
function deleteFriend(friendUserId) {
  return request({
    url: '/ranking/friend/delete',
    method: 'POST',
    data: { friendUserId },
  });
}

module.exports = { getRanking, deleteFriend };
```

## 四、需要你确认/补充的点

1. **avatarUrl 字段**：当前 ranking API 返回 `avatarUrl`，但前端 `ranking.wxml` 中使用的是 `avatar`（昵称首字）。如果你的用户头像已上传到微信，需要把 `avatarUrl` 正确显示出来。
2. **今日排行 calcTodayRank**：当前用 `app.globalData.userInfo?.openId` 匹配，如果本地没有存 openId 会返回 `null`。建议你在登录后把 `openId` 存到 `app.globalData.userInfo.openId`。
3. **SVG 图标**：你说要用 svg 文件夹下的图标，但我找了项目里没有 `miniapp/svg/` 这个目录。目前先用 emoji 代替了，你把 svg 文件放进去后告诉我，我来替换。
